# Device Configurations

This directory contains the running configurations of all network devices used in the Enterprise Network Lab.

## Files

| File | Description |
|------|-------------|
| router-running-config.txt | Cisco Router running configuration |
| switch-running-config.txt | Cisco Layer 2 Switch running configuration |

## Notes

These configurations correspond to Release **v1.0.0**.

They can be used to recreate the network topology or compare configuration changes in future releases.
