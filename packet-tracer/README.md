# Packet Tracer Files

This directory contains the Cisco Packet Tracer project files for the Enterprise Network Lab.

Each release represents the network at a specific development milestone.

---

## Current Version

**enterprise-network-lab-v1.5.pkt**

---

## Requirements

- Cisco Packet Tracer 8.2 or later

---

## Implemented Features

### Network Infrastructure

- Enterprise Network Topology
- VLAN Segmentation
- Router-on-a-Stick
- Inter-VLAN Routing
- DHCP
- Extended Access Control Lists (ACL)

### Network Management

- SSH Remote Management
- Dedicated Management VLAN
- Network Time Protocol (NTP)

### Network Security

- Port Security
- Switch Hardening
- Secure Remote Administration

### Layer 2 Technologies

- Spanning Tree Protocol (PVST)

---

## Documentation

The Packet Tracer topology corresponds to the documentation available in the `/docs` directory.

Each implemented feature includes:

- Configuration
- Design Decisions
- Verification
- Screenshots
- Troubleshooting Notes
