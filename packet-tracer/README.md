# Packet Tracer Files

This directory contains the Cisco Packet Tracer project files for the Enterprise Network Lab.

## Current Version

**enterprise-network-lab-v1.0.pkt**

## Requirements

- Cisco Packet Tracer 8.2 or later

## Features Implemented

- VLANs
- Router-on-a-Stick
- Inter-VLAN Routing
- DHCP
- Extended ACL
- Basic Enterprise Network Topology

## Notes

This Packet Tracer file corresponds to Release **v1.0.0**.

Future versions of the lab will be added to this directory while preserving previous releases.
