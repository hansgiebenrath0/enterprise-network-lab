# Project Overview

## Introduction

This project simulates a small enterprise network using Cisco Packet Tracer.

The objective is to design, implement, document, and troubleshoot a realistic business network while following networking best practices.

Unlike a typical Packet Tracer lab, this repository focuses not only on configuration but also on documentation, version control, and professional project organization.

---

## Project Scope

The current implementation includes:

- Enterprise network topology
- IPv4 addressing
- DHCP
- VLAN segmentation
- 802.1Q trunking
- Router-on-a-Stick
- Extended ACL

Future versions will expand the project by introducing Linux servers, Docker, Windows Server, Active Directory, VPN, and monitoring services.

---

## Learning Objectives

This project is intended to improve practical skills in:

- Enterprise networking
- Cisco IOS configuration
- Network troubleshooting
- Documentation
- Version control using Git and GitHub
