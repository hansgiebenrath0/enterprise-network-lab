# Network Diagrams

This directory contains the network diagrams for the Enterprise Network Lab.

## Files

| File | Description |
|------|-------------|
| topology.png | Network topology image |
| topology.drawio | Editable Draw.io diagram |

## Purpose

These diagrams provide a visual representation of the enterprise network architecture.

The editable Draw.io file allows future updates as the project evolves.
